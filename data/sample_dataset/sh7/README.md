## SH-7 dummy dataset (4 cases)

This folder contains 4 synthetic SH-7 “packets” to simulate a parent filing with attachments.

Each case folder contains:
- `SH7.md` (parent filing)
- `Board Meeting.md` (attachment)
- `EGM.md` (attachment)
- `MOA.md` (attachment)

All values are dummy/synthetic and are intended for testing document ingestion, classification,
parent-child linking, extraction, and traceability.
