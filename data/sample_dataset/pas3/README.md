## PAS-3 dummy dataset (4 cases)

This folder contains 4 synthetic PAS-3 “packets” to simulate a parent filing with attachments.

Each case folder contains:
- `PAS-3 Form.md` (parent filing)
- `List of Allottees.md` (attachment)
- `Board Resolution Allotment of Shares.md` (attachment)
- `Shareholder Undertaking.md` (attachment; extra supporting doc used for traceability tests)

All values are dummy/synthetic and are intended for testing document ingestion, classification,
parent-child linking, extraction, and traceability.
